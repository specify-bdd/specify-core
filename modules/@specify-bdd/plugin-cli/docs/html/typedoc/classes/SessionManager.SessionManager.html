    Class SessionManager

    Session Manager

    Manages the lifecycle of a system process session. Ensures only one command runs at a time, and tracks command output and exit code.

    run

    • run(command: string, opts?: SessionManagerOptions): CommandMeta

      Executes a single command within a managed session.

      Only one command may be active at a time in any given session. Output and exit code are available via output and exitCode once resolved.

      Parameters

      • command: string

        The command to execute

      • opts: SessionManagerOptions = {}

        Options to modify the behavior of run()

      Returns CommandMeta

      Remarks

      Multiple commands can be chained in a single command string with "&" or ";". Ex: echo first;echo second

      Throws

      AssertionError If another command is already in progress
